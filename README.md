# 🚀 agent-toolkit

**Modular, state-safe, and resource-efficient tools for modern AI Agents.**

## 📖 About
This toolkit bridges the gap between Large Language Models (LLMs) and real-world data. Unlike standard implementations, it prioritizes **linear logic flow** and **minimal memory footprint**, making it ideal for both high-scale deployments and lightweight personal assistants like Smirc/Aira.

## ✨ Key Features
- **Smart Location Tracking:** Distance-based updates (>5km) to save resources.
- **Resilient Web Search:** Multi-instance SearXNG support with automatic health checks.
- **Context-Aware Time & Weather:** Accurate local time and weather data based on user location.
- **LLM-Ready Architecture:** Seamless integration with Function Calling protocols.
